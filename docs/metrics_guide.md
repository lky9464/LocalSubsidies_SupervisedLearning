# 평가지표 쉽게 이해하기

이 문서는 데이터/통계 전문가가 아닌 분도 모델 결과를 이해할 수 있도록 작성했습니다.

## 위험도 점수(Risk Score)

- **범위**: 0 ~ 1000점
- **의미**: 점수가 높을수록 부정수급 위험이 높다고 모델이 판단한 것
- **만드는 방법**: 모델이 계산한 “위험(양성) 확률”에 1000을 곱해 반올림

## 혼동행렬(Confusion Matrix) 네 칸

실제와 예측을 네 가지로 나눕니다.

| 이름 | 뜻 |
|------|-----|
| TP (True Positive) | 실제로 위험(타겟)인데, 모델도 위험으로 맞춤 |
| TN (True Negative) | 실제로 정상이고, 모델도 정상으로 맞춤 |
| FP (False Positive) | 실제는 정상인데, 모델이 위험이라고 잘못 봄 (과탐) |
| FN (False Negative) | 실제는 위험인데, 모델이 정상이라고 놓침 (미탐) |

현장에서는 **미탐(FN)** 과 **과탐(FP)** 의 비용이 다를 수 있습니다.

## 주요 지표

| 지표 | 한 줄 설명 | 값이 높으면 |
|------|------------|-------------|
| 정확도(Accuracy) | 전체 중 맞춘 비율 | 전반적으로 많이 맞춤 (불균형 데이터에서는 해석 주의) |
| 정밀도(Precision) | “위험”이라고 한 것 중 진짜 위험 비율 | 잘못 집어내는 비율이 낮음 |
| 재현율(Recall) | 진짜 위험 중 모델이 찾아낸 비율 | 놓치는 위험이 적음 |
| 특이도(Specificity) | 진짜 정상 중 정상으로 맞춘 비율 | 정상 건을 잘 걸러냄 |
| F1점수(F1) | 정밀도와 재현율의 균형 점수 | 둘을 고르게 잘함 |
| ROC-AUC | 점수 순서가 위험을 잘 구분하는지 (0.5=우연, 1=완벽) | 순위 능력이 좋음 |
| PR-AUC | 불균형(위험 건이 적을 때)에서 더 참고하는 곡선 아래 면적 | 소수 위험군 탐지에 유리 |

### 불균형 데이터란?

전체 중 위험(타겟) 비율이 매우 낮을 수 있습니다. 이때 정확도만 보면 “전부 정상”이라고 해도 높아 보일 수 있어, **재현율·정밀도·PR-AUC·상위 K% 리프트**를 함께 봅니다.

## 상위 K% 리프트(Lift)

위험도 점수가 높은 상위 1%, 5%, 10%를 “우선 점검 대상”으로 볼 때:

- **양성 비율** (= precision@topK): 그 구간 안에 실제 타겟이 얼마나 있는지
- **양성 포착 비율** (= recall@topK): 전체 타겟 중 상위 K%에 포함된 비중
- **리프트**: (그 구간 양성 비율) ÷ (전체 양성 비율)  
  - 예: 리프트 5 → 무작위보다 5배 더 많이 타겟을 포함

## 점수 구간별 타겟율

0~100, 100~200, … 구간마다 실제 타겟 비율을 봅니다.  
점수가 올라갈수록 타겟 비율이 함께 올라가면, 점수 체계가 현장 직관과 맞습니다.

> 건수가 매우 적은 구간(기본 n<5)은 개인 식별 위험을 줄이기 위해 수치를 가릴 수 있습니다.

## 모델 비교 시 보는 순서 (추천)

1. ROC-AUC / PR-AUC (순위·불균형 성능)
2. 상위 5%~10% 리프트 (실제 점검 업무와 유사)
3. 재현율·정밀도 (놓침 vs 과탐 균형)
4. 혼동행렬 건수
